# Erikenobi Picks Instagram Landing

Este es el repo real de la landing publicada para Instagram.

## Qué contiene

- `index.html`: link tree principal
- `styles.css`: diseño mobile-first
- `script.js`: animaciones suaves
- `social-preview.svg`: preview social

## Objetivo de este repo

Servir únicamente como página corta de Instagram:

- canal free
- contacto directo por Telegram
- accesos rápidos a planes premium

## Qué no es este repo

Este repo no es:

- el bot principal
- el bot premium
- la landing comercial larga de ventas

## Separación recomendada

- `erikenobi_picks_landing`: link tree de Instagram
- repo o proyecto aparte: landing comercial larga
- repo del bot: automatización y lógica Telegram

## Publicación

Este proyecto está pensado para desplegarse desde la raíz del repo como sitio estático.
